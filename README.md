
# Playwright MCP sample (TypeScript)

Minimal sample project `playwright-mcp` demonstrating Playwright Test with TypeScript.

Prerequisites:

- Node.js 16+ installed

Install and run tests:

```bash
npm install
npx playwright install
npm test
```

Files of interest:

- `playwright.config.ts` — Playwright configuration in TypeScript
- `tests/example.spec.ts` — sample test in TypeScript

Useful scripts:

- `npm test` — run tests headless
- `npm run test:headed` — run tests with headed browser
- `npm run test:debug` — run tests in debug mode
